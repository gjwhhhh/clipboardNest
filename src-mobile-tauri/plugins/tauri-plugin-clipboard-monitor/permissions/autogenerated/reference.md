## Default Permission

Allows clipboard monitoring, read, and write commands.

#### This default permission set includes the following:

- `allow-start-monitoring`
- `allow-stop-monitoring`
- `allow-get-text`
- `allow-set-text`
- `allow-get-image`
- `allow-set-image`
- `allow-update-clipboard-content`
